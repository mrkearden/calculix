wsize 

   'wsize' [RETURN]|'f']|[<xp> <yp>]
Size of the window were xp and yp are the size in pixel. Without argument the size is the initial size and with the argument 'f' (fullsize) the screen resolution is used. This command takes ONLY effect during reading and execution of a batch file IF it is written in the very first line. Otherwhise it will be executed after reading of the batch file in the glut event loop (the glut library [2] for window management and event handling) which means that the program has to go into the interactive mode to take effect.

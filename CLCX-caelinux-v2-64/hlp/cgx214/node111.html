mats 

   'mats'
This keyword is used to get an overview of all material numbers defined. New sets will be created for all material numbers of elements. This feature is usually used if a mesh with mateial numbers was read. For example if a native-netgen file [4] was read then this command will generate sets which can be used to asign boundary conditions.
